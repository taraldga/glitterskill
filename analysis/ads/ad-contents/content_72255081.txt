Vår ambisjon er å bli den foretrukne plattformen for små og mellomstore mediehus og for å få til det trenger vi en person som kan hjelpe oss med å utvikle og forbedre nettstedene for nye kunder Du bør derfor være en som er interessert i å jobbe med alt fra databaser til CSS Samtidig som du liker å snakke med kunder Dette skal du jobbe med Du vil jobbe tett sammen med de seks andre utviklerne våre for å bygge en driftsikker og skalerbar arkitektur og et grensesnitt som elskes av brukerne Vi har kontroll over hele koden selv så du vil få jobbe med både backend og frontend Du brenner for å bygge gode nettløsninger og kjenner godt til språk som PHP MySQL Javascript HTML og CSS samt Linux som operativsystem Det er flott om du også er interessert i å leke med ny teknologi og rammeverk som React Node og RequireJS Samtidig håper vi at du er en person som setter pris på å kunne veksle mellom både store og små oppgaver;
